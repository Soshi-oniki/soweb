今回作るゲーム

今回は、ボタン連打ゲームをenchant.jsで作っていきます！

1.はじめに･･

Enchant.jsは、JavaScriptというプログラミング言語をゲーム作成に特化させた言語で、 誰でも簡単にゲームを制作することができます。
ここでは、Enchant.jsの基本的な書き方を学びながら、簡単なゲームを制作していきます。
まずは、初期設定のコードから書いていきましょう。 


        //初期設定
        window.focus();
        enchant(); // enchant.js を有効化

        window.onload = function() 
        {
            var game = new Game(320, 320); // ゲーム画面のサイズを指定
            game.fps = 30; // フレームレート（1秒間のフレーム数）
            game.preload('button.png'); // 画像を事前読み込み（今回はボタンの画像）

            game.onload = function() 
            {
                //ここより下にコードを記述
            };
            game.start(); // ゲームスタート
        };

上に示したコードがenchant.jsの基本となります。 画像の読み込みに使用する名前は自由なので、自分がわかる名前にしておくことをおすすめします！

2.スコアラベルの表示

ここからいよいよ、本格的なプログラミングをしていきます！ 


        game.onload = function() {
            var score = 0; //変数「スコア」を0にする。
            var label = new Label("Score: " + score);
            label.x = 10; //x座標指定
            label.y = 10; //y座標指定
            //メインシーンにラベル（この場合はスコア）を表示する。
            game.rootScene.addChild(label); 
        };

ここで、最終行の、

game.rootScene.addChild(label);
について少し解説します。
すべてのプロジェクト（スプライトやラベルなど）は、シーンに追加することで画面に表示されます。
そして、上のコードを使うことで、スプライトやラベルを画面に追加できます。
